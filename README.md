# Daftar Package
- expo-camera
- expo-location

# Cara Install dan Menjalankan
```bash 
    git clone https://github.com/n0l3r/my-fav-place.git
    cd my-fav-place
    npm install
    npm start
```

# Cara Build menggunakan Expo
```bash
    expo build:android
    expo build:ios
```

# Cara Build menggunakan eas
```bash
    eas build --platform android
    eas build --platform ios
```
