<h1> My Favs Place </h1>
<p> Aplikasi ini berguna untuk mengambil dan menyimpan foto beserta informasi lokasi yang diambil. </p>

# Daftar Package
- expo-camera
- expo-location

# Cara Menjalankan
```bash 
    git clone https://github.com/n0l3r/my-fav-place.git
    cd my-fav-place
    npm install
    npm start
```
